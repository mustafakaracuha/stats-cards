# Stats Cards 


## 👨🏻‍💻 Technologies and Libraries

- React Js
- Framer Motion
- Tailwind CSS
- React Icons
- Material UI

  
## ⚙️ Installation 

```bash 
    git clone https://github.com/mustafakaracuha/stats-cards.git
```
```bash 
    cd stats-cards
```
```bash 
    npm install
```
```bash 
    npm run dev
```
